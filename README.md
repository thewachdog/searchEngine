# searchEngine
<hr>
A search engine which is used for search in private networks.
<br>
#### Technologies Used:
<br>
- Python3
- Apache Nutch (For Crawler)
- Apache Solr (For Indexing)

<br>
#### Installation:
<br>
Clone this repo <br>
```git clone https://github.com/thewachdog/searchEngine.git```

<br>
Navigate into the folder <br>
```cd searchEngine```
